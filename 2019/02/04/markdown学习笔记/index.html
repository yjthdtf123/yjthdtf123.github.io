什么是markdown

Markdwon 是一种轻量级标记语言，它以纯文本形式（易读、易写、易更改）编写文档，并最终以HTLM格式发布。Markdown也可以理解为将以MARKDOWN语法编写的语言转换成HTML内容的工具。

markdown 语法

1.标题

在markdown中想要设置标题，则在标题的文字前添加#，一个#代表一级标题，两个# 代表二级标题，以此类推，只支持到六级标题即######，七个#号以上就是一段普通的文本。

注意：严谨的来说，设置标题时，#后边一般要加上空格。有的编辑器不加#这个符号也可以，但是最好全部加上空格，这样是最安全的。

2.字体

markdown中可以对字体进行强调，如加粗，斜体，斜体加粗，删除线等。

  • 加粗

要加粗的文字左右两端用**号包裹住

  • 斜体

要倾斜的文字左右两端用*号包裹

  • 斜体加粗
    要倾斜加粗的文字左右两端用***号包裹

  • 删除线
    要加删除线的文字左右两端用~~号包裹

9.代码

在markdowm中插入其他语言的代码使用反引号`将代码包裹

语法：

单行代码: 代码的开始和结束部位用反引号`包裹

代码块：代码之间用三个反引号包裹起来，并且两边的反引号单独占一行

示例：

单行代码： int a = 10; System.out.println("hello world");

代码块：

注： 在使用代码块时，可以在开始的`后面加上某一个编程语言名称，代码块会自动对代码进行高亮显示，看起来更方便。

例如要写一个java语言的代码块，那么开端就应该是`java，然后这个代码块中的java代码会自动的进行高亮显示。
